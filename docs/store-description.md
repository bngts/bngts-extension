방통실(bngts.com)에서 멀티뷰를 사용할 때, 로그인이 안 되어서 채팅을 못 치거나 플레이어가 불편했던 적 있으신가요?

방통실 Plus는 치지직과 SOOP의 로그인을 멀티뷰 환경에 자동으로 연동해주고, 플레이어를 최적화해주는 확장 프로그램입니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💬 로그인 자동 연동 - 멀티뷰에서도 채팅 참여

멀티뷰 사이트에서는 보안 정책 때문에 각 플랫폼의 로그인 상태가 연동되지 않습니다. 방통실 Plus를 설치하면 이 문제가 해결됩니다.

• 치지직에 로그인되어 있다면 멀티뷰에서도 자동으로 로그인 상태 유지
• SOOP에 로그인되어 있다면 멀티뷰에서도 자동으로 로그인 상태 유지
• 별도의 추가 로그인 없이 바로 채팅 참여 가능
• 구독 전용 채팅방이나 슬로우 모드 해제 등 로그인 사용자 기능 이용 가능

이제 멀티뷰에서도 좋아하는 스트리머의 채팅에 자유롭게 참여하세요!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ 플레이어 자동 최적화 - 더 넓은 화면으로 시청

멀티뷰 환경에 맞게 플레이어가 자동으로 최적화됩니다.

[치지직]
• 와이드 모드 자동 전환으로 영상 영역 최대화
• 채팅창 헤더 자동 접기로 깔끔한 화면
• 선호 화질 자동 선택 (1080p/720p/480p/360p)

[SOOP]
• 임베드 모드에서 불필요한 UI 요소 숨김
• 화질 선택 기능 활성화
• 선호 화질 자동 선택 (1440p/1080p/720p/540p/360p)
• 다크 모드 자동 적용으로 눈의 피로 감소

설정에서 각 플랫폼의 선호 화질을 지정하면 방송 시작 시 자동으로 해당 화질로 재생됩니다.

여러 방송을 한 화면에 배치할 때 각 플레이어의 공간을 최대한 활용할 수 있습니다.

📋 스트리머 즐겨찾기 관리

자주 보는 스트리머를 등록해두고 편리하게 관리할 수 있습니다.

• 닉네임이나 ID로 스트리머 검색
• 검색 결과에서 프로필 사진, 현재 방송 여부 확인
• 최대 50명까지 즐겨찾기 등록
• 드래그 앤 드롭으로 원하는 순서대로 정렬
• 전체/온라인/오프라인 필터링
• 이름순, 라이브 우선순 정렬

등록한 스트리머 중 시청할 스트리머만 체크하고 '시청하기' 버튼을 누르면 방통실 멀티뷰 페이지로 바로 이동합니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔔 방송 시작 알림

좋아하는 스트리머가 방송을 시작하면 놓치지 않도록 알려드립니다.

• 등록한 스트리머가 방송을 시작하면 데스크톱 푸시 알림
• 알림에 방송 제목 표시
• 알림 클릭 시 해당 스트리머의 방송 페이지로 바로 이동
• 브라우저 툴바 아이콘에 현재 방송 중인 스트리머 수 배지 표시
• 설정에서 알림 기능 켜기/끄기 선택 가능

바쁜 일상 중에도 좋아하는 스트리머의 방송 시작을 놓치지 마세요!

🌐 사용 방법

1. 확장 프로그램을 설치합니다
2. 치지직이나 SOOP에 평소처럼 로그인합니다
3. 방통실(bngts.com)에서 멀티뷰를 사용합니다
4. 로그인이 자동으로 연동되어 채팅 참여가 가능합니다!

스트리머 즐겨찾기 기능을 사용하려면:
1. 브라우저 툴바의 방통실 Plus 아이콘을 클릭합니다
2. 스트리머를 검색하여 추가합니다
3. 시청할 스트리머를 체크하고 '시청하기'를 클릭합니다
